Zombie-Arena игра в которой игрок движется по арене и убивает зомби (вид сверху).
В игре:
-Можно перезаряжать оружие;
-Подбирать патроны;
-Подбирать аптечки;
-Зомби имеют 3 типа скоростные охотники с наименьшим количеством жизней, среднескоростные толстяки со средним количеством жизней и обрубки с наибольшим количеством жизней;
Игра выложена в архиве Release Zombie-Arena.zip и приложена картинка игры Zombie-Arena.jpg
A,W,D,S -перемещение игрока
R -перезарядка
пробел(Space) - стрельба
мышкой или тачпадом наведение на цель
